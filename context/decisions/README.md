# ADR

Каталог содержит только действующие architecture decision records. Пропуски в нумерации намеренные: superseded и
чисто исторические ADR удаляются по принятой в проекте политике отсутствия исторического шума, а оставшиеся не
перенумеровываются. Архив удаленных решений — git history; номера удаленных ADR не переиспользуются.
